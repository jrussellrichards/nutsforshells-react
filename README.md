# Aplicación para Nutsforshells

## NutsForShells

NutsForShells es un emprendimiento familiar de venta e intercambio de moluscos de todo el mundo.

* Pagina web: http://nutsforshells.com/

## Contexto

La empresa trabaja con datos de miles de especies en donde cada una de estas tiene una pecularidad o característica que la diferencia. Es por esto que el manejo de datos e inventario se ha hecho muy complicado, sobre todo debido al rápido crecimiento de la empresa. 

## Solución

Desarrollo de una aplicación web que permita manejar el inventario, con las operaciones de, actualizar, eliminar y agregar nuevos datos. Además se añaden características solicitadas por la empresa como, posibilidad de descargar listas en pdf, filtrar datos en tiempo real entre otras.

## Tecnología utilizada

* React Js
* Express Js
* Mysql
